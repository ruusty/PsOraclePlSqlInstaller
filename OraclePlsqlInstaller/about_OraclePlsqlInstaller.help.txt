SHORT DESCRIPTION
    Cmdlets used execute Pl/Sql files based on file naming coventions
           
LONG DESCRIPTION
    Cmdlets used execute Pl/Sql files based on file naming coventions
                 
    The following cmdlets are included.

        ConvertTo-SqlPlusCommands
        Get-OracleSecret
        ConvertTo-SqlPlusCommands.ps1
        Get-BuildList
        New-Fixtures
        New-OracleRunFixture
        Set-OracleUserPassword
        Set-SdlcConnections
        Show-OracleSecret
        Test-OracleConnections


    Get all the cmdlets in  OraclePlsqlInstaller
    get-module "OraclePlsqlInstaller" | select -expand ExportedCommands

    Get the help

    $(get-module OraclePlsqlInstaller).ExportedCommands.Keys |% {get-help $_}

    The Date folder contains boilerplate and example files.
